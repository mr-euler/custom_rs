
# Самописный РС кодер/декодер

## Использование

### Скрипт с примерами

```bash
make main
./main
```

```bash
./build.sh main
./main
```

### Скрипт с кодером

```bash
make coder
./coder
```

```bash
./build.sh coder
./main
```

### Удаление исполняемых файлов

```bash
make clean
```
